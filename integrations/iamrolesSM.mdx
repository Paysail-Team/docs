To enhance security, we've created a custom policy that provides the minimum permissions necessary for Syro to synchronize secrets with the AWS Parameter Store. You can apply this policy to the IAM user as follows:

### Policy

1. Navigate to the Create New Policy section in the AWS IAM console
2. Switch to the JSON tab
3. Enter the following policy

```bash
{
      "Version": "2012-10-17",
      "Statement": [
      {
          "Sid": "AllowSecretsManagerAccess",
          "Effect": "Allow",
          "Action": [
               "secretsmanager:GetSecretValue",
               "secretsmanager:DescribeSecret",
               "secretsmanager:PutSecretValue",
               "secretsmanager:CreateSecret",
               "secretsmanager:DeleteSecret",
               "secretsmanager:TagResource",
               "secretsmanager:UpdateSecret
            ],
            "Resource": "*"
         }
     ]
}
```
4. Name your policy and press create
